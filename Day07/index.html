3.Css Background Size

  1. background-size: contain; keyword syntax ("auto", "cover" and "contain")
  2. background-size: 500px; one-value syntax (sets the width of the image & height becomes "auto")
  3. background-size: 75% 50%; two-value syntax (first value: width of the image, second value: height)
